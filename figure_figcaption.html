Тег <figure>

HTML тег <figure> является блочным элементом, его содержимое представляет собой самостоятельный контент, который связан с основным содержимым страницы. Обычно он содержит картинки, диаграммы, фотографии, фрагменты кода или схемы, которые упоминаются в основном тексте, но которые без труда можно перенести на другую страницу или в приложение без ущерба для основного контента. Элемент <figure> может содержать внутри себя элемент <figcaption>, который позволяет добавить подпись к контенту, он должен располагаться в качестве первого либо последнего дочернего элемента

Синтаксис

<figure>
…
</figure>

Пример

<figure>
<img src="masterpiece.jpg" alt="Шедевр" >
<figcaption>Един с природой</figcaption>
</figure>

Результат данного примера в окне браузера

Элемент <figcaption> является необязательным и может появляться до или после содержимого внутри </figure>. Только один элемент <figcaption> может быть помещен в </figure>, хотя сам элемент </figure> может содержать несколько дочерних элементов (например, <img> или <code>).

Преимущество и недостатки

Преимуществом данно тега является то,что контент страницы не пострадает при его перемещении или удалении. При этом тег <figure> может использоваться не только для изображение,но и для фрагментов кода, аудио или видо и т. д. Так же есть возможность вложить один блок figure в другой, если это имеет смысл. Если нужно добавить больше смысла к картинке, то можете использовать h1 и p в figcaption.
В качестве недостатка можно отметить то, что изображение, вставленное с помощью данных тегов не может располагаться внутри абзаца, то есть нет обтикаемости текстом.
